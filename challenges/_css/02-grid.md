## Descripción

Se te solicita maquetar la vista de una tienda online básica que cuenta con Menú de navegación, una barra lateral con la cantidad de productos por categoría, una grilla de productos y un pie de página.

## Objetivo 

Tu tarea es construir el **layout del ecommerce usando CSS Grid** y su propiedad **grid-template-areas**, para mantener el layout de referencia.

Ejemplo:

![Demo reto Ecommerce]({{ site.url }}{{ site.baseurl }}/assets/images/demo-reto-ecommerce.webp)

## Requerimientos

1. Crear un layout con **CSS Grid** y su propiedad **grid-template-areas**.
2. Crea una grilla de productos con **CSS Grid**.
3. Utilizar **Flexbox** para distribuir los elementos de la barra lateral.
4. Utilizar **Flexbox** para la organización de las tarjetas de productos.
