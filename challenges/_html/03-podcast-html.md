```md
## Objetivo

El objetivo es aprender a integrar contenido multimedia correctamente, estructurar la información del episodio y ofrecer una experiencia clara al usuario.

Debes utilizar la etiqueta **<audio>** y organizar el contenido con etiquetas semánticas.

## Indicaciones

- Usar correctamente la etiqueta **<audio>**.
- Estructurar un episodio o programa de podcast
- Aplicar HTML semántico para contenido multimedia
- Pensar la página como un producto real (título, descripción, episodio, autor, etc.)
```

## Boilerplate

Usa **únicamente este HTML inicial** y construye el resto desde cero:

```html
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Reto 03 — Podcast en HTML</title>
</head>
<body>

  <header>
    <h1></h1>
    <p></p>
  </header>

  <main>
    <section>
      <h2></h2>

      <audio controls>
        <source src="" type="audio/mpeg">
        Tu navegador no soporta audio.
      </audio>

      <p></p>
    </section>
  </main>

  <footer>
    <p></p>
  </footer>

</body>
</html>
```
